# Guía de Inicio Rápido

## ✅ El servidor está corriendo correctamente

**URL de acceso:** http://localhost:3001

## 📝 Pasos para probar la aplicación

### 1. Abre tu navegador web
   - Chrome, Firefox, Edge, o cualquier navegador moderno

### 2. Ingresa a la URL
   ```
   http://localhost:3001
   ```

### 3. Verás la pantalla principal con 4 pestañas:

#### 🔵 **Usar Equipo**
   - Aquí puedes iniciar el uso de un equipo
   - Selecciona un equipo de la lista desplegable
   - Ingresa tu nombre
   - Describe el propósito del uso
   - Presiona "Iniciar Uso"

#### 🟢 **Equipos en Uso**
   - Muestra todos los equipos que están siendo usados actualmente
   - Puedes finalizar el uso desde aquí
   - Se actualiza automáticamente cada 5 segundos

#### 🟡 **Historial**
   - Ver todos los registros históricos
   - Buscar por usuario, equipo o propósito
   - Ver fechas de inicio y fin de cada uso

#### 🟣 **Gestionar Equipos**
   - Agregar nuevos equipos al laboratorio
   - Ver la lista completa de equipos
   - Ver el estado (disponible/en uso) de cada equipo

## 🎯 Prueba rápida

1. **Ve a "Usar Equipo"**
2. **Selecciona:** Microscopio Óptico
3. **Usuario:** Tu nombre
4. **Propósito:** Prueba del sistema
5. **Click en:** Iniciar Uso
6. **Verás una notificación verde** confirmando el registro
7. **Ve a "Equipos en Uso"** para ver tu registro activo
8. **Finaliza el uso** agregando observaciones opcionales

## 🔧 Para detener el servidor

En la terminal donde está corriendo, presiona:
```
Ctrl + C
```

## 🚀 Para volver a iniciar

```bash
npm start
```
